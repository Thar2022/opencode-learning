# OpenCode Notes

## บทที่ 1: ติดตั้ง & เริ่มต้น — 2026-05-09
- OpenCode = AI coding agent แบบ open-source ใช้ใน terminal, desktop app, IDE
- `/connect` = เชื่อมต่อ LLM provider (Claude, GPT, Gemini, ฯลฯ)
- `/init` = ให้ AI วิเคราะห์ project → อธิบายว่า project มีอะไรบ้าง → สร้าง AGENTS.md
- คำสั่งแรก: ถาม AI ว่า "อธิบาย project นี้หน่อย" ก็คือการ `/init` นั่นเอง
- ติดตั้งผ่าน: curl script, npm, brew
